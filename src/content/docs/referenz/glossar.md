---

## A

**AI Sichtbarkeit**
: Wie gut eine Website in Antworten von KI-Assistenten wie ChatGPT, Perplexity oder Claude zitiert wird.

## L

**LLM (Large Language Model)**
: Große Sprachmodelle wie GPT-4, Claude oder Llama die Text verstehen und generieren können.

**LLMO (LLM-Optimierung)**
: Die Optimierung von Websites für KI-Systeme, analog zu SEO für Suchmaschinen.

**llms.txt**
: Eine Textdatei im Root-Verzeichnis einer Website die speziell für KI-Systeme geschrieben ist.

## P

**Perplexity**
: Ein KI-Assistent der das Web durchsucht und Fragen mit Quellenangaben beantwortet.

## S

**Schema.org**
: Ein standardisiertes Vokabular für strukturierte Daten im Web.
